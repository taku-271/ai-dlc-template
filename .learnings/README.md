# .learnings

Claude がセッションをまたいで学びを蓄積するディレクトリです。
Gitで管理することでチーム全員が共有できる「チームの記憶」になります。

| ディレクトリ | 内容 |
|---|---|
| `errors/` | 発生したエラーと解決策 |
| `patterns/` | このプロジェクト固有のコードパターン・慣習 |
| `preferences/` | 人間からの指摘・好み・判断基準 |
| `context/` | プロジェクト固有の技術的背景・設計の前提知識 |

## ファイル命名規則
- `errors/[エラーの種類].md`
- `patterns/[パターン名].md`
- `preferences/[テーマ].md`
- `context/[テーマ].md`

## ファイルフォーマット

```markdown
## 概要
（何についての学びか）

## 内容
（具体的な内容・解決策・パターン）

## 発見日
YYYY-MM-DD

## 関連ファイル
- `src/...`
```
